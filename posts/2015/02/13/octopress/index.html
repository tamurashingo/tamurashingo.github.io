curl -sSL https://get.rvm.io | bash -s stable

環境変数を読み込み

source ~/.rvm/scripts/rvm

Rubyのインストール

rvm install 2.1.5

OK!

Octopressのインストール

git clone git://github.com/imathis/octopress.git octopress
cd octopress
gem install bundler
bundle install
rake install

これで実行しようとするとJSランタイムがないと言われるので、Gemfileに追加します。

vi Gemfile

こんな感じでtherubyracerを追加します。

  gem 'rubypants', '~> 0.2.0'
  gem 'rb-fsevent', '~> 0.9'
  gem 'stringex', '~> 1.4.0'
  gem 'therubyracer'
end

rake previewでプレビューを確認します。以上！
