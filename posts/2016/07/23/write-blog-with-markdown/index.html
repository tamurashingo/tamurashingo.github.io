Markdownでブログを書く

tamuraです。

以前はGhostでブログを書いており、その時はMarkdown形式で書いていました。 livedoor blogなどのいわゆる一般向けのブログサービスはMarkdown形式でブログを書けないため、ここ数日いろいろと検討して、いい感じにブログが書けるようになりました。

  1. Markdownでブログを書く (Visual Studio Codeがいいかも)
  2. markedでHTMLに変換する
  3. livedoor blogにHTMLタグ編集モードで書き込む

これでやっています。

Markdownでブログを書く

今までと同じようにMakrdown形式でブログを書いていきます。 プレビューが使えるため、私はVisual Studio Codeを使って書いています。

画像を貼る場合はわかりやすいように

[ここにVisualStudioCodeのキャプチャを貼る](ここにURL)

のように書いておいて、あとから何とかします。

まあここはあまり特筆することはないです。

markedでHTMLに変換する

WindowsでもMacでもLinuxでも、とりあえずnode.jsを入れて

npm install -g marked

でmarkedを入れておきます。

そして

marked blog.md -o blog.html

でMarkdownをHTMLに変換します。

livedoor blogにHTMLタグ編集モードで書き込む

変換後のblog.htmlをテキストエディタで開きます。 で、lviedoor blogをHTMLタグ編集モードにしてコピペします。

以上。
